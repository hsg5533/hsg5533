## 💻 About Me

안녕하세요!  
저는 **TypeScript**를 기반으로 **React** 및 **React Native**를 주력으로 개발하는 프론트엔드 개발자입니다.  
웹과 모바일 환경 모두에서 사용자 중심의 UI/UX를 구현하는 데에 열정을 가지고 있습니다.

- **Frontend:** React, React Native, TypeScript
- **Server-side (Sub):** Node.js, Spring Boot, Django
